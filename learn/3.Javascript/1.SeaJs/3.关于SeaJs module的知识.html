Seajs构建工具

使用条件

a. exports === module.exports

b. exports 是 module.exports 的引用

c. exports 只能扩展属性，不能直接赋值

/**错误写法
*  该对象由于是引用关系，不能直接进行赋值
**/
exports = {a:a}

/**正确写法**/
module.exports = {a:a}

/**解析
*
*  由于exports 与 module.exports 是引用关系
*  exports.a = a 相当于在被引用的对象上添加属性再赋值
*  exports = {a:a} 相当于 var exports = new Object() 再添加属性和值的的简写，这是一个新对象，引用的是新地址，与module引用地址不同
*  
**/
